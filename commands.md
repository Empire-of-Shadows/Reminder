# Bump Commands
* /bump setup - Interactive setup menu for initial configuration
* /bump settings - Manage all bump reminder settings (includes premium activation)


# Broadcast Related commands
* /alerts subscribe - Toggle your broadcast subscription (run once to join, again to leave)
* /alerts status - Check your broadcast subscription status
* /broadcast manage - Interactive menu to create and manage broadcasts
* /broadcast send - Send a one-time broadcast announcement
