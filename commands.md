# Admin Commands
* /admin panel - Unified configuration panel (Components v2). Covers Core Setup
  (bump channel, bump role, timers channel), Bump Bots (enabled bots + per-bot
  cooldowns), Messages (custom reminder text + timer embed toggle), and Premium
  (status + code activation). Requires the Manage Server permission.


# Misc Commands
* resync_guild


# Owner-only (DM)
* !admin - Hidden premium code tooling (generate / view / deactivate codes).
